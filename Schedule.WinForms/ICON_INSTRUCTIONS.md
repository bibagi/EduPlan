# Инструкция по добавлению иконки приложения

## Где разместить файл иконки

Поместите файл иконки с именем `icon.ico` в папку:
```
Schedule.WinForms/icon.ico
```

## Требования к файлу иконки

- **Формат**: `.ico` (Windows Icon)
- **Рекомендуемые размеры**: 
  - 16x16 пикселей
  - 32x32 пикселей
  - 48x48 пикселей
  - 256x256 пикселей
- **Цветность**: 32-bit (с прозрачностью)

## Как создать .ico файл

### Вариант 1: Онлайн конвертер
1. Создайте изображение в формате PNG (рекомендуется 256x256 пикселей)
2. Используйте онлайн-конвертер, например:
   - https://convertio.co/ru/png-ico/
   - https://www.icoconverter.com/
3. Загрузите PNG и скачайте готовый .ico файл

### Вариант 2: Photoshop/GIMP
1. Создайте изображение нужного размера
2. Сохраните как .ico файл

### Вариант 3: Visual Studio
1. Добавьте в проект новый элемент "Icon File"
2. Отредактируйте иконку в встроенном редакторе

## После добавления иконки

1. Поместите файл `icon.ico` в папку `Schedule.WinForms/`
2. Пересоберите проект:
   ```
   dotnet build Schedule.WinForms/Schedule.WinForms.csproj
   ```
3. Иконка появится:
   - В заголовке окна приложения
   - В панели задач Windows
   - В файле .exe приложения
   - В ярлыках приложения

## Текущая структура проекта

```
Schedule.WinForms/
├── icon.ico                    ← Поместите иконку сюда
├── Schedule.WinForms.csproj    ← Уже настроен для использования icon.ico
├── Forms/
├── Helpers/
└── ...
```

## Примечание

Если файл `icon.ico` отсутствует, приложение будет использовать стандартную иконку Windows.
После добавления иконки не забудьте пересобрать проект!
