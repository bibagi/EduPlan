📁 В этой папке находятся примеры файлов для импорта данных
   в систему управления расписанием.

═══════════════════════════════════════════════════════════

📋 ФОРМАТЫ ФАЙЛОВ:

1. УЧИТЕЛЯ (Учителя.txt)
   Формат: Полное имя [TAB] Краткое имя
   Пример:
   Иванов Иван Иванович	Иванов И.И.
   Петрова Мария Сергеевна	Петрова М.С.

2. АУДИТОРИИ (Аудитории.txt)
   Формат: Название [TAB] Вместимость
   Пример:
   101	30
   Спортзал	50

3. ПРЕДМЕТЫ (Предметы.txt)
   Формат: Название предмета (каждый с новой строки)
   Пример:
   Математика
   Русский язык
   Физика

4. ГРУППЫ (Группы.txt)
   Формат: Название [TAB] Курс
   Пример:
   ИС-21	2
   ПО-31	3

═══════════════════════════════════════════════════════════

🔧 ПОДДЕРЖИВАЕМЫЕ РАЗДЕЛИТЕЛИ:

- TAB (табуляция) - рекомендуется
- ; (точка с запятой)
- , (запятая)

Примеры:
Иванов Иван Иванович	Иванов И.И.
Иванов Иван Иванович;Иванов И.И.
Иванов Иван Иванович,Иванов И.И.

📊 ИМПОРТ ИЗ EXCEL:

Также поддерживается импорт из файлов .xlsx
Формат:
- Первая строка - заголовки (игнорируется)
- Со второй строки - данные
- Каждый столбец = одно поле

Пример для учителей:
| Полное имя              | Краткое имя  |
|-------------------------|--------------|
| Иванов Иван Иванович    | Иванов И.И.  |
| Петрова Мария Сергеевна | Петрова М.С. |

═══════════════════════════════════════════════════════════

💡 КАК ИСПОЛЬЗОВАТЬ:

1. Откройте приложение
2. Войдите как администратор
3. Откройте нужный справочник (Учителя, Аудитории и т.д.)
4. Нажмите кнопку "Импорт"
5. Выберите файл из этой папки
6. Следуйте инструкциям

═══════════════════════════════════════════════════════════

⚠️ ВАЖНО:

- Дубликаты не импортируются (показывается ошибка)
- Пустые строки игнорируются
- Неправильный формат = ошибка для этой строки
- После импорта показывается отчёт: успешно/ошибок
